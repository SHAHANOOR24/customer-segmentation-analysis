# Customer Segmentation & Shopping Trends Analysis

## Project Overview

This is a client-based analytics project focused on understanding customer shopping behavior using a retail dataset of **3,900+ records** from the U.S. market.

The objective was to analyze customer spending patterns and perform **customer segmentation using machine learning techniques** to derive business insights and improve targeted marketing strategies.

## Tools & Technologies

* Python
* MySQL
* Pandas
* NumPy
* Matplotlib / Seaborn
* Scikit-learn
* Jupyter Notebook

## Problem Statement

* Identify product categories where discounts should be applied
* Analyze spending behavior by age, location, and season
* Segment customers into 3 or more groups
* Recommend targeted marketing campaigns

## Key Analysis Performed

* Customer spending analysis by category
* Age-wise and location-wise spending trends
* Seasonal purchase behavior
* Payment method analysis
* Customer clustering using machine learning

## Key Insights

* Identified high-value customer groups
* Recognized seasonal and discount-driven buyers
* Derived insights for targeted promotions and retention campaigns
